# LINAC - Normativas Gráficas

## Logotipo

El logotipo combina tipografía con iconografía de estrella. Elementos:
- **LINAC** en tipografía bold
- **ACTITUD LINGÜÍSTICA** como descriptor secundario
- Separador vertical entre ambos elementos

### Área de Seguridad
Mantener espacio libre equivalente al tamaño de la estrella del logo en todos los lados.

### Reducción Mínima
Cuando el logo mida menos de 1cm (o tamaños muy pequeños en web), usar únicamente el isotipo "LINAC" sin el descriptor.

## Tipografías

### Logotipo
- **Branding SF w05** (solo para el isotipo, no usar en textos)

### Títulos y Cuerpo
- **Bahnschrift** para títulos y textos de cuerpo
- Fallback web: Segoe UI, sans-serif

## Variaciones del Logo

### Positivo
Logo con "LINAC" en azul corporativo (#233a63) sobre fondos claros o blancos.

### Negativo
Logo completo en blanco sobre fondos oscuros o de color.

### Horizontal
Usar en espacios con limitación vertical (headers, banners).

### Imagotipo
Versión simplificada para favicon, redes sociales, marcas de agua.

## Valores de Marca (Visuales)

Usar etiquetas angulares con colores corporativos:
- **Educación** → Azul corporativo
- **Lenguaje** → Verde
- **Inclusión** → Morado
- **Online** → Azul oscuro

## Restricciones

- Respetar siempre el área de seguridad del logo
- No distorsionar las proporciones del logotipo
- No usar el logo sobre fondos que comprometan su legibilidad
- Preferir versión negativa sobre gradientes azul-verde de marca
- No usar colores fuera de la paleta oficial en elementos de identidad

## Aplicaciones Web

### Gradientes Permitidos
Combinar azules corporativos con verdes para fondos de secciones destacadas, banners o hero sections.

### Contraste
Siempre verificar contraste WCAG AA para texto sobre fondos de color.
